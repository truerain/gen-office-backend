# 공통 팝업 API

## 사용자 검색 팝업
사용자 선택이 필요한 화면에서 공통으로 사용하는 사용자 검색 API.

### Endpoint
- `GET /api/common/users`

### Query params
- `q` (optional): 사용자명 부분 검색 (empName 기준)

### Response
```json
[
  {
    "userId": 1001,
    "empNo": "E1001",
    "empName": "홍길동",
    "titleCd": "c",
    "titleName": "책임",
    "orgId": "ORG001",
    "orgName": "인사팀"
  }
]
```

### Notes
- `q`가 없으면 전체 사용자 목록을 반환한다.
